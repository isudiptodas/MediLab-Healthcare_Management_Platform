<h2>Doctor Appointment Booking</h2>
<p><b>Tech Stack : </b> <i>React, MongoDB, Express, NodeJS, BullMQ, Redis, Node-Cron, Arcjet</i></p>

<br/>

<p><b>Features :</b></p>
<ul>
  <li>Email Validation</li>
  <li>Role based access control</li>
  <li>Email reminder for appointments</li>
  <li>Account recovery with email based OTP</li>
  <li>AI health consultation</li>
</ul>
